MEAN Belt Exam

Description

Given 4.5 hours to complete the exam. I was expected to create a full stack MEAN project and deploy it. The prompt was Polls. I had to create a web application that allowed users to sign-in and create/delete polls with four options and answer other polls created by other users. Polls can only be deleted by people with the same name. There is also a logout function which clears a user's name.

Features

  • Unauthenticated Login
  • Form Validations
  • Displaying all polls from Database
  • Allowing users to view and vote on polls
  • Use of CRUD operations
  • Was Deployed on AWS

Technologies

  • MongoDB
  • Node.js
  • Express.js
  • Angular4
  • JavaScript
  • AWS
