Python Belt Exam

Description

Given 4.5 hours to complete the exam. I was expected to create a full stack django project and deploy it. The prompt was Friends. I had to create an application that allowed users to login/register and add or remove other users as friends. I also had to add a view user profile function.

Features

  • Unique Login / Registration
  • Form Validations
  • Displaying all users from Database
  • Allowing users to friend other users
  • Allowing users to view other user's profile
  • Use of CRUD operations
  • Password Encryption
  • Was Deployed on AWS

Technologies

  • Python
  • Django
  • SQLite
  • Bcrypt
  • Regex
  • AWS
